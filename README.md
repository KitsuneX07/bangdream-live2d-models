# BanG Dream! live2d models

提取自游戏`BanG Dream! Girls Band Party!`的所有live2d模型。来源为日服。最近更新日期为2025.12.16

脚本批量输出，因此不保证100%可用性。不过从日志输出来看，错误率应该不大于**0.23%**。建议遇到文件错误配合[Bestdori!](https://bestdori.com/)食用。

可以使用Live2DViewerEX查看和编辑模型，可以将模型部署在个人博客等支持的平台。

模型仅供交流学习使用，禁止用于任何商业用途。

不提供具体的教程，请有能力者自行检索教程。

推荐使用[gitzip](https://gitzip.org/)等工具下载对应文件夹而非克隆整个仓库(~4+GB)

文件索引可以在此处找到[index](index.json)
